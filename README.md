# Anti-Bot
## This bot will ban all users that sends a message in the #antibot channel.

## How to use?
1. Add the bot to your server.
2. Create a channel named `#antibot`. Any user (except other bots) who sends a message here will be banned.
3. (Optional) Create a channel named `#antibot-logs`. The bot will post ban reports there with embeds.
4. The bot automatically deletes 7 days of message history for any banned user.

## Why
Discord has a problem, more and more server are victims of scam bot accounts, joining their server and spamming @everyone ping + scam images or some adults stuff server invite links.

## How
This bots will ban ALL account that sends a message in the Anti-Bot channel.
